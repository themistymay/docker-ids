# Intrusion Detection Systems (IDS)
## Splashed with Docker Goodness

### Requirements
* docker-compose

### Build
#### Base Image
``` docker build -t themistymay/ubuntu:java8 java/ ```

#### Stack
``` docker-compose up ```
